# modelo_conceitual
O modelo conceitual de banco de dados é uma representação abstrata de alto nível do mini-mundo, focando nas regras de negócio e nas entidades, atributos e relacionamentos. Independente de tecnologia (SGBD), utiliza diagramas Entidade-Relacionamento (DER) para definir "o que" será armazenado, servindo de base para modelos lógicos e físicos.
